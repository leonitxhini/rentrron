# Logo Ordner

Platzieren Sie hier Ihr Logo:

- **logo.png** oder **logo.svg** - Hauptlogo (empfohlen: SVG für beste Qualität)
- **logo-dark.png** oder **logo-dark.svg** - Logo für dunkle Hintergründe (optional)
- **logo-light.png** oder **logo-light.svg** - Logo für helle Hintergründe (optional)
- **favicon.ico** - Favicon für den Browser

**Empfohlene Größen:**
- Logo: Mindestens 200x60px (oder höher für Retina-Displays)
- Favicon: 32x32px oder 16x16px

Nach dem Hinzufügen des Logos wird es automatisch in der Navbar und anderen Komponenten verwendet.
