# Hero Bild Ordner

Platzieren Sie hier Ihre Hero-Bilder:

- **hero-head.png** oder **hero-head.jpg** - Das Bild für die Hero-Section (Head Section)
- **hero-image.png** oder **hero-image.jpg** - Das Bild für die "Best Customer Experience" Section
- Empfohlene Größe: Mindestens 1920x1080px für beste Qualität

Nach dem Hinzufügen der Bilder werden sie automatisch verwendet.
